# Docker Практическое задание

## Ответы на вопросы

#### 1. Какая версия Docker установлена на вашей системе?
Docker version 27.5.1, build 9f9e405

#### 2. Сколько слоев в hello-world образе?
   2 слоя

#### 3. Какой размер hello-world образа?
   20.3kB

#### 4. Что делает команда CMD в hello-world?
   CMD ["/hello"] запускает исполняемый файл /hello, который выводит "Hello from Docker!"

#### 5. Почему hello-world такой размер?
   Содержит только статически скомпилированный исполняемый файл, без ОС и библиотек

#### 6. Какая разница между docker pull и docker run?
   docker pull - только скачивает образ, docker run - скачивает и запускает контейнер

#### 7. Где хранятся Docker образы на вашей системе?
   /var/lib/docker

#### 8. Что показывает docker info о вашей системе?
   16 CPU, 15.54GiB RAM, x86_64, Docker Desktop на WSL2, 7 контейнеров, 6 образов

#### 9. Сколько контейнеров hello-world вы запустили?
   2 контейнера
